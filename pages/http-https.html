HTTP & HTTPS (โปรโตคอลรับส่งเว็บ)

HTTP (Hypertext Transfer Protocol) คือโปรโตคอลสำหรับรับส่งข้อมูลเว็บ ส่วน HTTPS คือ HTTP ที่เพิ่มการเข้ารหัสด้วย SSL/TLS เพื่อความปลอดภัย

เมื่อผู้ใช้เปิดเว็บเบราว์เซอร์ คำขอจะถูกส่งผ่าน HTTP หรือ HTTPS ไปยังเซิร์ฟเวอร์ เว็บส่วนใหญ่ในปัจจุบันใช้ HTTPS เพื่อปกป้องข้อมูลผู้ใช้
