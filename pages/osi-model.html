OSI Model (โมเดลโอเอสไอ)

OSI Model เป็นกรอบงาน (framework) ที่แบ่งการสื่อสารเครือข่ายออกเป็น 7 ชั้น เพื่อให้เข้าใจและพัฒนาได้ง่าย

  1. Physical Layer (ชั้นกายภาพ) จัดการกับสัญญาณไฟฟ้าและสายเคเบิล
  2. Data Link Layer (ชั้นเชื่อมโยงข้อมูล) ดูแลการส่งข้อมูลเป็นเฟรม
  3. Network Layer (ชั้นเครือข่าย) จัดเส้นทางและ IP Address
  4. Transport Layer (ชั้นขนส่ง) ดูแลการเชื่อมต่อและการรับส่งแบบ TCP/UDP
  5. Session Layer (ชั้นเซสชัน) จัดการการสร้างและสิ้นสุดเซสชัน
  6. Presentation Layer (ชั้นการนำเสนอ) แปลงข้อมูลให้เหมาะสม เช่น การเข้ารหัส
  7. Application Layer (ชั้นแอปพลิเคชัน) เป็นจุดที่โปรแกรมใช้งานเครือข่าย เช่น HTTP
